# Omni-SRE Landing Page — Design System

## Brand
- **Name**: Omni-SRE — The AI-first Site Reliability Engine
- **Personality**: Premium, futuristic, intelligent, trustworthy, powerful

## Colors
| Token | Value |
|---|---|
| Background Light | `#ffffff` |
| Background Dark | `#050510` |
| Primary Accent | `#3b82f6` (Electric Blue) |
| Secondary Accent | `#7c3aed` (Violet) |
| Success | `#10b981` (Emerald) |
| Danger | `#ef4444` (Crimson) |
| Text Primary | `#0a0a14` |
| Text Secondary | `#6b7280` |
| Glass Card | `rgba(255,255,255,0.08)` + `backdrop-blur` |

## Typography
- **Font**: Inter (Google Fonts) — weights 300, 400, 600, 700, 800, 900
- Hero heading: 96px, weight 900, letter-spacing -0.04em
- Section heading: 56px, weight 800
- Body: 18px, weight 400, line-height 1.7

## Effects
- Magnetic cursor: 60px radial gradient orb, blue-to-violet
- Particle field: 120 floating dots, connected by lines within 150px
- 3D card tilt: perspective 1000px, rotateX/Y up to 15deg
- Glassmorphism: white/10 bg, backdrop-blur-xl, 1px white/20 border
- Scroll animations: fade-up translateY(40px), IntersectionObserver threshold 0.15
- Gradient orbs: large blurred circles, blue + violet
- Smooth scrolling: scroll-behavior smooth

## Buttons
- Primary: #0a0a14 bg, white text, pill shape, hover scale 1.05
- Ghost: transparent, 1px white border, pill shape, hover fill white
- CTA glow: primary + blue box-shadow glow on hover
